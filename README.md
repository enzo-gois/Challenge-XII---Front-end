## Getting Started

Como Rodar:

```bash
# Clone o repositório
git clone https://github.com/enzo-gois/Challenge-XII---Front-end

# Navegue até o diretório do projeto
cd Challenge-XII---Front-end

# Instale as dependências
npm install
```

Acesse a página  ->   http://localhost:3000
